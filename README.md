
# WonderLust

A brief description of what this project does and who it's for


## Run Locally

Clone the project

```bash
  git clone https://github.com/TarunPatil001/wonderlust.git
```

Go to the project directory

```bash
  cd wonderlust
```

Install dependencies

```bash
  npm i express
  npm i mongoose
  npm i ejs
```

Start the server

```bash
  nodemon app.js
```
